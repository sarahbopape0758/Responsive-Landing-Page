# Task 01 – Responsive Landing Page  

This project is an **interactive responsive landing page** with a **fixed navigation bar** that changes color when you scroll and adds hover effects when you move over the menu items.  
It was created using **HTML, CSS and JavaScript**.

---

##  Live Demo

View the website here:  
 https://sarahbopape0758.github.io/Responsive-Landing-Page/


---

##  Project Structure


Everything was built in **one main file** for simplicity and easy deployment.

---

##  Technologies Used

- HTML5 – structure of the page
- CSS3 – design, colors, layout, hover effects
- JavaScript – interaction (scroll & navigation changes)

---

##  Features

-  Fixed navigation bar (always visible)
-  Navbar changes background when scrolling
-  Hover effects on menu items
-  Responsive layout for all screen sizes
-  Clickable email & portfolio links
-  Smooth and clean layout
